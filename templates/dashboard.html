📬 텔레그램 알림 설정

카페 업로드 진행 상황을 텔레그램으로 알림 받습니다.
업로드 성공/실패/대기 알림이 실시간으로 전송됩니다.

Bot Token (@BotFather에서 발급)
Chat ID (개인 또는 그룹)
💡 텔레그램 봇 설정 방법
1. 텔레그램에서 @BotFather 검색 → /newbot 명령
2. 봇 이름 설정 후 발급된 Bot Token을 위에 입력
3. 생성된 봇에게 아무 메시지 전송
4. Chat ID 확인: 브라우저에서 https://api.telegram.org/bot{TOKEN}/getUpdates 접속 → chat.id 값 복사
5. 테스트 전송 버튼으로 확인
